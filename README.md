# ⌨️ TypeTest - Typing Speed Test

A modern and interactive **Typing Speed Test** application built with **React + Vite**. Test and improve your typing speed with real-time character highlighting, WPM calculation, accuracy tracking, and a beautiful animated UI.

## 🚀 Live Demo

🔗 **Live Website:** https://typetestchecker.netlify.app/

## ✨ Features

- 📝 Random typing paragraphs fetched from an external API
- ⏱️ 60-second countdown timer
- 🚀 Real-time Words Per Minute (WPM) calculation
- 🎯 Live typing accuracy tracking
- 🟢 Character-by-character highlighting:
  - Green → Correct character
  - Red → Incorrect character
  - Cursor indicator for current character
- 📊 Dynamic progress bar
- 🔄 Restart test with a new random paragraph
- 🎨 Modern glassmorphism UI with animated gradient background
- 📱 Fully responsive design for desktop and mobile

## 🛠️ Tech Stack

- **React.js**
- **Vite**
- **JavaScript (ES6+)**
- **CSS3**
- **Bacon Ipsum API** (for random paragraphs)

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/<your-github-username>/<your-repository-name>.git
```

Navigate to the project folder:

```bash
cd <your-repository-name>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
